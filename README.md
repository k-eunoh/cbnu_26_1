## 🗂️ 카테고리별 요약

### 🔌 API
Flask 기반 REST API 설계 및 구현. Stateless/Stateful 구조 비교, RESTCONF를 활용한 네트워크 장비 제어.

### ⚙️ Automation
Python, Ansible, Bash, PowerShell, Terraform을 활용한 인프라 자동화. 반복 작업의 코드화 및 자동 배포.

### 🌐 Networking
Cisco, Aruba, Ubiquoss, AhnLab 장비의 실무 설정 문서. Layer2/3 구성, 보안, VPN, 무선망 구축.

### 🖥️ Server
HP/HPE, Dell/EMC, Lenovo, Synology 서버 하드웨어 구성 및 운영 가이드.

### 🐧 Linux OS
Ubuntu/CentOS 기반 네트워크 설정, 서비스 구성, 시스템 관리.

### 🪟 Windows OS
Windows Server 네트워크, AD, 서비스 구성.

### 🔮 Hypervisor OS
VMware ESXi, Proxmox VE 가상화 플랫폼 구성 및 운영.

### 🏆 Skills Competition
IT네트워크 시스템 직종 기능경기대회 연도별 과제 분석 및 솔루션 (2017~2024).
