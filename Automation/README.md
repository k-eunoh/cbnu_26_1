## 핵심 개념

- **Ansible**: 에이전트리스 자동화 도구. YAML 기반 Playbook으로 Linux/Windows/네트워크 장비 일괄 제어
- **Terraform**: 선언형 IaC 도구. 클라우드/온프레미스 인프라를 코드로 정의·배포
- **EEM (Embedded Event Manager)**: Cisco IOS 내장 자동화 엔진. 이벤트 발생 시 자동 조치
- **Python**: Netmiko, Paramiko, NAPALM 등 라이브러리로 네트워크 장비 원격 제어
