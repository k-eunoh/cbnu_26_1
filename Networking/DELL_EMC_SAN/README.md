# DELL/EMC SAN

> Dell/EMC SAN 스토리지 네트워크 설정 기술문서

## SAN 개요

- **SAN (Storage Area Network)**: 서버와 스토리지를 전용 네트워크로 연결
- **FC (Fibre Channel)**: 전통적 고속 SAN 프로토콜
- **iSCSI**: IP 네트워크 기반 SAN (비용 효율적)
- **FCoE**: Ethernet 기반 Fibre Channel

## 적용 사례

- 괴산군 BIS 스토리지 이중화
- 고가용성 스토리지 구성
