# AhnLab

> AhnLab 보안 장비 설정 기술문서 — 방화벽, UTM

## 적용 사례

- 괴산군 BIS 외부 공격 방어
- 사내 인프라 보안 강화

## 주요 기능

| 기능 | 설명 |
|------|------|
| 방화벽 (Firewall) | 정책 기반 트래픽 제어 |
| IPS/IDS | 침입 탐지 및 방지 |
| VPN | IPSec / SSL VPN |
| 애플리케이션 제어 | L7 트래픽 제어 |
| 로깅/모니터링 | 보안 이벤트 기록 |
