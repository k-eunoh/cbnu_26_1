# Aruba

> HPE Aruba 무선 AP 및 컨트롤러 설정 기술문서

## 적용 사례

- 충북 스쿨넷 무선망 분리 사업
- 괴산군 BIS 무선망 구축

## 주요 제품

| 제품군 | 설명 |
|--------|------|
| Aruba AP | 무선 액세스 포인트 |
| Aruba Controller | 중앙 집중 관리 컨트롤러 |
| Aruba Central | 클라우드 기반 관리 플랫폼 |
| ClearPass | 네트워크 접근 제어 (NAC) |

## 무선 보안 설정

- **WPA3 Enterprise** — 802.1X 인증
- **SSID 분리** — 업무망 / 게스트망 분리
- **VLAN 매핑** — SSID별 VLAN 할당
- **RF 최적화** — 채널·전력 자동 조정
