## 직종 소개

**IT 네트워크 시스템** 직종은 다음 영역을 포함한다:

- **네트워크 구성** — 스위칭, 라우팅, 보안, 무선
- **서버 구성** — Linux/Windows Server, 가상화
- **서비스 구성** — DNS, DHCP, Web, Mail, FTP
- **자동화** — 스크립트, 모니터링
