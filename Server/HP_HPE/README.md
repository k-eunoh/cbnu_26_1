# HP / HPE Server

> HP/HPE 서버 하드웨어 설정 및 운영 문서

## 주요 제품

| 제품 | 설명 |
|------|------|
| ProLiant DL 시리즈 | 랙마운트 서버 |
| ProLiant ML 시리즈 | 타워 서버 |
| iLO (Integrated Lights-Out) | 원격 관리 카드 |
| SmartArray | RAID 컨트롤러 |

## iLO 원격 관리 주요 기능

- KVM (원격 콘솔)
- 가상 미디어 마운트
- 전원 관리 (시작/중지/재시작)
- 하드웨어 상태 모니터링
- 이벤트 로그 (IML)
