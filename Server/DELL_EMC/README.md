# Dell / EMC

> Dell/EMC 서버 및 스토리지 설정 기술문서

## 적용 사례

- 괴산군 BIS 스토리지 이중화 및 백업
- 사내 IDC 서버 구축

## RAID 구성

| RAID 레벨 | 최소 디스크 | 용도 |
|-----------|------------|------|
| RAID 0 | 2 | 성능 (데이터 보호 없음) |
| RAID 1 | 2 | 미러링 (OS용) |
| RAID 5 | 3 | 성능 + 보호 (일반 서버) |
| RAID 6 | 4 | 이중 패리티 (스토리지) |
| RAID 10 | 4 | 고성능 + 보호 |

## iDRAC 주요 기능

- Virtual Console (KVM)
- 가상 미디어 마운트
- 전원 관리
- 시스템 이벤트 로그 (SEL)
- RAID 구성 및 모니터링
