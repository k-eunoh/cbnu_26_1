# Synology NAS

> Synology NAS DSM 설정 및 운영 기술문서

## 적용 사례

- 사내 통합 스토리지 / 백업 시스템

## 주요 기능

| 기능 | 설명 |
|------|------|
| 파일 서버 | SMB / NFS / AFP |
| 백업 | Hyper Backup, Active Backup |
| RAID 관리 | SHR, RAID 1/5/6/10 |
| 클라우드 동기화 | Cloud Sync (Azure, S3) |
| 컨테이너 | Container Manager (Docker) |
